WeClawBot

微信连接的黑白屏电子记事贴。把一句留言、提醒、问候或读书笔记安静地留在桌面、餐桌和床头。

Open firmware for a WeChat-connected e-paper sticky note display, built for the Waveshare ESP32-S3-RLCD-4.2.

安装 / 配置 中文说明 English README

微信输入

扫码连接后，在微信里发送文字，设备自动更新屏幕；微信端也可以继续修改、清除或替换内容。

浏览器安装

固件刷写和 Wi-Fi 配置走本地浏览器 USB 页面，适合公开仓库直接分发。

隐私可审计

仓库只公开设备端固件和用户指引，云端整理服务可配置，用户可选择自己的 provider。
